                  在数学中，辗转相除法，又称欧几里得算法（Euclidean algorithmEuclidean\ algorithmEuclidean algorithm），是一种求取最大公约数的算法。出现于欧几里得的《几何原本》 ，其原理就是两个数的最大公约数等于其中较小的数字和二者之间余数的最大公约数，通过这种算法，就可以求出最大公约数。

                  # 辗转相除法是怎么用的

                  首先来看两个例子：

                  例一 已知两个整数198198198，366366366，求它们的最大公约数：
                  解 因为198<366198<366198<366，所以先用366366366除以198198198，得到111，余数为168168168
                  接着，用198198198除以168168168，得到111，余数为303030
                  然后，用168168168除以303030，得到555，余数为181818
                  然后，用303030除以181818，得到111，余数为121212
                  之后，用181818除以121212，得到111，余数为666
                  最后，用121212除以666，余数为000
                  所以198198198和366366366的最大公约数是666

                  例二 已知两个整数169776169776169776，347787347787347787，求它们的最大公约数：
                  解 因为169776<347787169776<347787169776<347787，所以先用347787347787347787除以169776169776169776，得到222，余数为823582358235
                  接着，用169776169776169776除以823582358235，得到202020，余数为507650765076
                  然后，用823582358235除以507650765076，得到111，余数为315931593159
                  然后，用507650765076除以315931593159，得到111，余数为191719171917
                  之后，用315931593159除以191719171917，得到111，余数为124212421242
                  然后，用191719171917除以124212421242，得到111，余数为675675675
                  然后，用124212421242除以675675675，得到111，余数为567567567
                  之后，用675675675除以567567567，得到111，余数为108108108
                  之后，用567567567除以108108108，得到555，余数为272727
                  最后，用108108108除以272727，得到444，余数为000
                  所以169776169776169776与347787347787347787的最大公约数是272727

                  # 辗转相除法的通项公式

                  通过 上述过程，我们可以得出，辗转相除法就是先用较大数除以较小数，然后不断用除数除以余数，知道除尽为止，得出的的最后一个余数就是两数的最大公约数
                  最大公约数的英文是 GreatestCommonDivisorGreatest Common DivisorGreatestCommonDivisor，简称 gcdgcdgcd，因此通项公式可以写为：

                  gcd(a,b)=gcd(b,b mod a)   (b>a)gcd(a,b)=gcd(b,b\ mod\ a)\ \ \ (b>a)gcd(a,b)=gcd(b,b mod a)   (b>a)

                  # 证明方法

                  《几何原本》里是用几何角度证明的， 在这里采用几何方式证明

                  # 几何证法

                  证明 设有长度分别为 aaa、bbb 的两条线段 ABABAB、CDCDCD ，且 AB<CDAB<CDAB<CD
                  用 ABABAB 量取 CDCDCD ，量到剩下一个 EFEFEF
                  接着，用 EFEFEF 量取 CDCDCD ，量到剩下一个 GHGHGH
                  最后，用 GHGHGH 量取 EFEFEF ，刚好量完
                  此时，GHGHGH 可以量取 EFEFEF ，同样能量取自身，所以它可以量取 EF+GHEF+GHEF+GH ,也就是 ABABAB ，也能够量取 CDCDCD
                  即 GHGHGH 的长度是 ABABAB、CDCDCD 的整数倍
                  即 GHGHGH 的长度是 aaa、bbb 的最小公约数
                  （您可以使用Geogebra帮助理解）

                  # 代码实现

                  有了理论基础，代码实现就很简单了，首先可以写一个递归，让他用除数一直除以余数，直到余数为0为止，最后返回最后一个余数就行了

                  int gcd(int a,int b) {
    if(a % b == 0) return b;
    return gcd(b, a%b);
}

                  当然这个代码还能继续压缩，变成三目运算符形式

                  int gcd(int a,int b) {
    return a%b ? gcd(b, a%b) : b;
}

                  # 实战演练

                  # 洛谷B3736 最大公约数open in new window

                  这个题很简单，求三个数的最大公约数，只要先求出两个数的最大公约数，再与第三个求就行了，代码如下：

                  #include<iostream>
using namespace std;
//gcd模板
int gcd(int a,int b) {
    return a%b ? gcd(b, a%b) : b;
}
int main() {
	int a,b,c;
	scanf("%d%d%d",&a,&b,&c);
	printf("%d\n",gcd(gcd(a,b),c)); 
	return 0;
}

                  # 洛谷B3634 最大公约数和最小公倍数open in new window

                  两个数求最大公约数和最小公倍数，也是很简单，最大公约数用上面的最大公约数模板可以求得，而最小公倍数就是两数之积除以两数的最大公约数，注意虽然两数在int范围，但最小公倍数可能超过int范围，代码如下：

                  #include<iostream>
using namespace std;
//gcd模板
long long gcd(long long a,long long b) {
    return a%b ? gcd(b, a%b) : b;
}
int main() {
	long long a,b;
	scanf("%lld%lld",&a,&b);
	printf("%lld %lld\n",gcd(a,b),a*b/gcd(a,b)); 
	return 0;
}

                  我们也可以写出最小公倍数(LeatestCommonMultiple LCM)(Leatest Common Multiple\ LCM)(LeatestCommonMultiple LCM)模板

                  int lcm(int a,int b) {
    return a*b/gcd(a,b);
}

                  # 提示

                  gcd()和__gcd()的区别在于后者是经过编译器优化过的，在某些平台下会有更好的性能，而gcd()是C++17标准中在<algorithm>头文件中的函数，在C++11下不可用
